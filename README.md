# JustLog
Log anything you want via command
lightweight stand alone Plugin

## How it works ?
### First
 - Put the Plugin in the Plugins Folder
### Second
 - Config How you want it to work
### Third
 - enjoy

## Command
 - `/justlog (filename/secondfilename/...) what you want to log %thatsupportpapi%

## Permission
 - `justlog.admin` to use the command (False by default)

## Supported storage
 - text (default)
 - yaml
 - json
 - csv
 - sqlite
 - ~~mysql~~ (todo)

### Plugin Supported
 - PlaceholderApi ([Spigot](https://www.spigotmc.org/resources/placeholderapi.6245/)/[Github](https://github.com/PlaceholderAPI/))
 - any plugin that you can use command in it

and you are done
